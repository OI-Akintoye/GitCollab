Social Media

For example, Facebook was formed in the mid-2000s and has grown to be one of the biggest social media platforms, amounting to over a billion users. It allows people to send to each other friend requests and private messages, publish posts which can be seen by a range of people, which can be chosen by the user posting, among many other things.

It was the only one like it at the time. This excludes blogs, which do allow for comments to be posted but are not as centralised and easy to connect with others as Facebook or Instagram.
